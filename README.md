Система для оформления заявок на получение справок сотрудниками.

Desctop-приложение, реализованное на WPF для создания и обработки запросов сотрудников для получения справок. 

Функциональность:
- сотрудник может создать заявку;
- сотрудник может просматривать свои заявки, в том числе их статус;
- бухгалтер может просматривать заявки от всех сотрудников;
- бухгалтер может изменять статус заявки сотрудника.

Запуск программы:
1. Скачайте архив TestTask_release.rar и распакуйте его на своем устройстве.
2. В папке publish найдите файл TestTask.exe и запустите его.

При первом запуске автоматически создаётся база данных SQLite и выполняется заполнение тестовыми данными.
